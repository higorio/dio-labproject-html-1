O que é HTML

HTML - HyperText Markup Language, diferente do que muitos pensam, não é uma lingaguem de progração utilizada no lado do cliente e nem do servidor, mas é uma lingagem que como o próprio nome diz: Linguagem de Marcação de Hipertextos. Ela serve justamente para definir a estrutura das páginas e aplicações web que a gente navega por ai através de suas tags de marcação. voltar

O que são tags

Uma tag é estrutura de código predefinidos utilizada para construção de páginas HTML. Normalmente uma página é construída com a marcação de várias tags.

Uma tag pode receber atributos que são propriedades que são passadas paera a tag para controlar seu funcionamento.

Exemplos de nomes de tags:

  • h1
  • h2
  • h3
  • h4
  • h5
  • h6
  • p
  • ul
  • li
  • a
  • hr
  • small
  • strong

Como funciona uma tag?

Toda tag inicia com sinal de menor do teclado, seguido pelo nome da tag e depois com o sinal de maior do teclado.
